## 二、支持的布局模式

根据图片数量自动匹配，也可手动切换：

| 每页图片数 | 布局模式 | 说明 |
|------------|----------|------|
| **1 张** | 全页 | 单图铺满整页 |
| **2 张** | 左右 / 上下 / 对角 | 二分页面 |
| **3 张** | 横排三等分 / 竖排三等分 / 两小一大 | 三分页面 |
| **4 张** | 四宫格 / 田字格 | 四等分页面 |
| **6 张** | 2×3 网格 / 3×2 网格 | 六宫格 |
| **9 张** | 3×3 网格 | 九宫格 |

## 三、功能清单

### 3.1 图片管理

| 功能 | 说明 |
|------|------|
| 添加图片 | 点击「添加」打开文件对话框，支持 PNG/JPG/WEBP/BMP |
| 移除图片 | hover 图片项 → 点击 × 按钮移除单张 |
| 移除最后一张 | 点击「移除」按钮 |
| 图片排序 | 拖拽图片到目标位置调整顺序 |
| 跨页移动 | 拖拽图片到缩略图区域的目标页面 |

### 3.2 页面设置

| 设置项 | 选项 |
|--------|------|
| 纸张大小 | A4 / A3 / Letter |
| 页面方向 | 纵向 / 横向 |
| 每页图片数 | 自动 / 1 / 2 / 3 / 4 / 6 / 9 |
| 图片间距 | 0-30mm 滑块调节（仅「有间距」模式） |
| 间距模式 | 有间距（图片带间距） / 铺满（edge-to-edge 无边距） |

### 3.3 预览交互

| 功能 | 说明 |
|------|------|
| 翻页 | 点击左右箭头或缩略图切换页面 |
| 交换图片位置 | 按住图片拖到另一个图片的位置 |
| 调整图片显示区域 | 铺满模式下按住图片拖动，调整裁切位置 |
| 调整分区大小 | 拖动相邻图片之间的边界线，调整各区域比例 |

### 3.4 导出

| 格式 | 说明 |
|------|------|
| **PDF** | 导出为 PDF 文件，保留所有调整 |
| **Word** | 导出为 .docx 文件，保留所有调整 |

## 四、间距模式对比

**有间距模式 (gapped)：**
- 图片周围有白色边距
- 图片使用 `object-contain` 完整显示
- 间距大小可调 (0-30mm)

**铺满模式 (edge-to-edge)：**
- 图片铺满整个页面区域，无边距
- 图片使用 `object-cover` 裁切填充
- 可拖动调整图片在裁切区域内的显示位置
