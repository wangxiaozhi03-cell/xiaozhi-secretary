# UI 主题设计规范

## 设计理念

整体 UI 采用 **HarmonyOS NEXT 沉浸光感（Immersive Light）设计语言**。

设计目标：

- 简洁
- 自然
- 轻盈
- 舒适
- 专业
- 现代
- 高级感

避免传统后台管理系统风格，整体更接近系统级应用，具有良好的视觉层次和长时间使用的舒适性。

---

# 整体风格

整体采用：

- 浅色主题（默认）
- 高留白设计
- 卡片式布局
- 扁平化设计
- 柔和阴影
- 大圆角
- 微动效
- 简洁布局

界面元素之间保持充足留白，减少视觉压力。

---

# 配色规范

## 主背景色

```
#DAF4F6
```

作为整个应用的默认背景。

视觉柔和、干净，符合 HarmonyOS 沉浸光感风格。

---

## 卡片背景

```
#FFFFFF
```

或

```
rgba(255,255,255,0.85)
```

支持轻微毛玻璃效果。

---

## 主文字颜色

```
#2B2F36
```

用于正文内容。

避免使用纯黑色，提高阅读舒适度。

---

## 次级文字

```
#5F6B76
```

用于说明文字、副标题、辅助信息。

---

## 弱化文字

```
#98A2B3
```

用于提示信息或禁用状态。

---

## 主主题色

```
#3B82F6
```

用于：

- 按钮
- Tab
- 菜单选中
- Switch
- Checkbox
- Radio
- Progress

保持统一视觉语言。

---

## Hover 背景

```
#EEF4FF
```

鼠标悬停时使用。

---

## 选中背景

```
#E8F1FF
```

用于菜单、列表、Tab 等选中状态。

---

## 分割线

```
rgba(0,0,0,0.06)
```

采用弱分割设计，不使用明显边框。

# 圆角规范

窗口：

```
24px
```

卡片：

```
16px
```

按钮：

```
12px
```

输入框：

```
10px
```

标签：

```
999px
```

整体保持统一圆角风格。

---

# 阴影规范

默认阴影：

```
0 8px 24px rgba(0,0,0,0.08)
```

悬停阴影：

```
0 12px 32px rgba(0,0,0,0.12)
```

阴影柔和自然，不使用厚重阴影。

---

# 毛玻璃效果

支持适量使用毛玻璃设计。

推荐：

```
background: rgba(255,255,255,0.75);

backdrop-filter: blur(24px);
```

仅用于：

- 设置窗口
- 浮层
- 弹窗
- 工具面板

避免全局滥用。

---

# 动画规范

动画风格应自然流畅。

动画时长：

```
180ms ~ 220ms
```

缓动：

```
ease-out
```

Hover：

轻微高亮或阴影变化。

Click：

轻微缩放反馈。

页面切换：

淡入淡出 + 平滑过渡。

避免复杂和夸张动画。

---

# 图标风格

统一使用线性图标。

要求：

- 风格统一
- 简洁现代
- 不使用彩色图标
- 不混用不同设计风格

推荐尺寸：

```
18px
```

菜单图标：

```
20px
```

---

# 留白规范

页面边距：

```
24px
```

```
16px
```

```
12px
```

```
8px
```

通过留白提升整体高级感和阅读体验。

---

# 设计原则

整个项目保持统一设计语言：

- 简洁而精致
- 柔和而清晰
- 专业而现代
- 高留白
- 高一致性
- 高可扩展性

所有新增页面、组件和工具模块都必须遵循本主题规范，确保整体视觉风格统一，打造具有 HarmonyOS NEXT 沉浸光感特色的现代开发者工具。
